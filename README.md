## Run Locally

Clone the project

```bash
  git clone https://github.com/dhruv-bvpdev/todo-app-rn.git
```

Go to the project directory

```bash
  cd todo-app-rn
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
